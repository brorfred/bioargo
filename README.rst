bioargo
========

bioargo is a package for reading, processing, and plotting data
from bioargo floats. 

Installation
------------
    pip install https://gitlab.rsg.pml.ac.uk/python/intro_python.git
    
Example of usage:
-----------------
    import bioargo
    fl = bioargo.Float()
    fl.plot()

Command line usage
------------------
    scatterplot
